# Setup and Running Instructions for iPad

## Prerequisites
- An iPad with iOS 14 or later.
- Ensure you have the latest version of Xcode installed on your Mac.

## Setup Steps
1. **Clone the Repository:**
   - Open your terminal and use the following command:
     ```bash
     git clone https://github.com/nichapaamhoperay/revenuetax.git
     ```
   
2. **Open the Project in Xcode:**
   - Navigate to the project directory and open `revenuetax.xcodeproj` in Xcode.

3. **Configure the App:**
   - In Xcode, select your target device as the iPad.
   - Ensure that you have set the appropriate provisioning profiles and certificates to run on the iPad.

4. **Run the App:**
   - Click the 'Run' button in Xcode to build and deploy the app on your iPad.

## Running the App
- Once the app is deployed on your iPad, you can launch it from your home screen and test all the functionalities as intended.
